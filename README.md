# Frontier.cool test

## 檔案結構

    └── apps
        ├── Nested-tree
        └── Random-user

## 如何啟動專案

#### 請先確認 Node 版本為 `version 14.18+, 16+` 才能正常啟動專案

### 🚀 啟動專案 Random-user

1. 移動至該資料夾

```bash
$ cd apps/Random-user
```

2. 安裝所需 node_modules

```bash
$ npm i
```

3. 啟動

```bash
$ npm run dev
```
